NightBot Commands

Copy these commands into your NightBot command responses. Each command serves a specific purpose for managing your stream leaderboard.

Add User Command

Allows users to add themselves to the leaderboard.

Example: {!add 100/200} - 100 is min gain, 200 is max gain

Edit User Command

Allows users to edit their gain rates.

Example: {!edit 100/200} - Updates min/max gains

Example Command

Shows how to create custom upload command with gain values.

Tip: Add &rates=true to the end of the url to alter channel gain rates

Get Subscriber Count

Retrieves a user's current subscriber count.

Returns the subscriber count for the specified user

Get User Gains

Retrieves a user's current gain rate.

Returns the gain rate for the specified user

Get User Rank

Retrieves a user's current rank on the leaderboard.

Returns the rank position for the specified user

NightBot Connection

Connect to NightBot to enable live chat commands for your stream.
